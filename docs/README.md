## 关于我们

Peckot Studios 的想法由 Pectics 首次于2020年3月28日提出，经过不断的更名尝试、版本迭代、周目更新，最终形成了如今的 Peckot Studios，Peckot Studios 当下也并不是一个实际公认的组织，只是对未来发展的一个规划罢了。

Peckot Studios 的工作方向不固定，但随着时间的推移我们相信 Peckot Studios 会有它一技之长的！
